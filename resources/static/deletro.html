Deletreo


El deletreo en lenguaje de señas es la representación manual de las letras del alfabeto con las manos, utilizada para deletrear palabras, especialmente nombres propios, términos técnicos o palabras que no tienen una seña establecida. Este sistema se conoce como alfabeto dactilológico y varía ligeramente entre diferentes lenguas de señas.
